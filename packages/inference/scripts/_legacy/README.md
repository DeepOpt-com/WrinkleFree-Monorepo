# Legacy Conversion Scripts

These scripts are archived alternatives to the primary converter. Use `convert_checkpoint_to_gguf.py` instead.

| Script | Purpose | Status |
|--------|---------|--------|
| `convert_to_sglkernel.py` | Convert to sgl-kernel packed format | Superseded by GGUF |
| `convert_with_quantization.py` | Conversion with quantization options | Merged into main converter |
| `convert_and_cache.py` | Conversion with GCS caching wrapper | Use main converter + manual caching |

For current conversion workflow, see the main [CLAUDE.md](../../CLAUDE.md).
