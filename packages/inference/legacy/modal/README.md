# Legacy Modal Benchmarks

This directory contains deprecated Modal benchmark scripts.

**Status:** Deprecated (December 2025)

**Reason:** Benchmarking infrastructure moved to local/SkyPilot execution.

## Files

- `modal_cpu_bench.py` - CPU kernel benchmarks
- `modal_sglang_build.py` - SGLang BitNet benchmarks
- `modal_extended_benchmark.py` - Extended stress tests
- `modal_quick_benchmark.py` - Quick smoke tests
- `modal_native_kernel.py` - Native AVX kernel tests

## Migration

Run benchmarks locally or via SkyPilot instead.
